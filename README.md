# Prompt Optimizer Skill

将模糊、随意的自然语言转化为专业级 AI 提示词。具备诊断、补全、重构、抽象能力，而非简单润色。

## 核心特性

- **7 维诊断**：分析原始提示词在角色、上下文、约束、格式、示例、推理引导等维度的缺失
- **三层重构**：按任务复杂度选择骨架（Tier 1）→标准（Tier 2）→高级（Tier 3）结构
- **克制优先**：给模型原则而非模板，信任其根据内容自主组织输出的能力
- **反模式消除**：识别并消除"过度约束""扼杀自主组织"等常见问题

## 使用场景

当用户说以下任意关键词时触发：

## 设计哲学

1. **诊断而非批评**：指出缺失维度，不贬低原始输入
2. **补全而非重写**：填补空白，不改变用户核心意图
3. **克制而非过度**：给原则而非模板，信任模型能力
4. **实用而非学术**：输出可直接复制粘贴的生产级提示词

```
## 优化后提示词
你是一名资深科技产品文案专家，拥有10年SaaS产品营销经验。
请为新上线的AI代码助手工具撰写产品介绍。目标受众是中小型软件公司的技术负责人。
要求：1) 突出核心功能 2) 强调ROI 3) 使用专业但易懂的技术语言 4) 字数控制在500字以内。
输出结构由你根据内容自行组织，怎么讲清楚就怎么讲。

## 改进说明
- 角色具体化：从"写手"升级为"资深科技产品文案专家 + 领域经验"
- 上下文补全：明确产品类型、目标受众
- 约束精炼：4条核心原则（功能、ROI、语言、字数）替代模糊要求
- 输出信任：用"自行组织"替代死板模板，保留灵活性
```
